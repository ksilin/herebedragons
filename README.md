Here be dragons - exploring slick 3
========

Created for the Dresden Scala User Group 2016.2.29, updated for 2017.7.13

## Run application
To run application, call:
```
sbt run
```
If you wanna restart your application without reloading of sbt, use:
```
sbt re-start
```
## Run tests
To run tests, call:
```
sbt test
```
